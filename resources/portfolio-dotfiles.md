I mostly use the following tools:

* **Operating System**: Arch Linux
* **Window Manager**: I3WM
* **Text editor**: Neovim (with about 80 plugins)
* **Shell**: ZSH, Prezto
* **Terminal**: Termite
* **Search tool**: Ack/Ag, Fuzzy File Finder, Locate
